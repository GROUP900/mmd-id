# Haha! It's you!

A function to get the MeowMeowLand ID photo, input appearance description, output image Buffer.

```javascript
import { generate } from "@mmd-id/generator";

generate({
  background: "#3157E1",
  mood: "happy",
  race: {
    type: "bunny",
    size: "medium",
    coat: "#D9B391",
  },
});
```
